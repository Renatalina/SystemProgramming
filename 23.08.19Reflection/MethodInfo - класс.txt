MethodInfo - класс
 
Выявляет атрибуты метода и обеспечивает доступ к его метаданным.
Пространство имен:   System.Reflection
Сборка:  mscorlib (в mscorlib.dll)

Иерархия наследования
System.Object

Синтаксис
[SerializableAttribute]
[ClassInterfaceAttribute(ClassInterfaceType.None)]
[ComVisibleAttribute(true)]
[PermissionSetAttribute(SecurityAction.InheritanceDemand, Name = "FullTrust")]
public abstract class MethodInfo : MethodBase, _MethodInfo

Заметки
Класс MethodInfo представляет метод типа. Можно использовать MethodInfo для получения сведений о методе, который представляет объект и вызова метода объекта. Пример:
Метод видимость можно определить путем получения значения IsFamilyAndAssembly, IsFamilyOrAssembly, IsPrivate, и IsPublic Свойства.
Вы можете узнать, какие атрибуты применяются к методу, получая значение Attributes свойства или метода GetCustomAttributesметод.
Можно определить, является ли это универсальный метод открытого сконструированного универсального типа или закрытому сконструированного универсального типа, получая значения из IsGenericMethod и ContainsGenericParameters Свойства.
Можно получить сведения о параметрах метода и возвращаемый тип из GetParameters метод и ReturnParameter, ReturnType, и ReturnTypeCustomAttributes Свойства.
Можно выполнить метод экземпляра класса путем вызова Invoke метод.
Можно создать экземпляр MethodInfo представляющий сконструированный универсальный метод, представляющий определение универсального метода, вызывая MakeGenericMethod метод.
Можно создать экземпляр MethodInfo экземпляры путем вызова Type.GetMethods или Type.GetMethod метод, или путем вызова MethodInfo.MakeGenericMethod метод MethodInfo представляющий определение универсального метода.
Список неизменяемых условий для конкретных универсальных методов см. в разделе IsGenericMethod свойство. Список неизменяемых условий для других условий, используемых в универсальном отражении см. в разделе IsGenericType свойство.
